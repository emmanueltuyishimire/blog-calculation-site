# **App Name**: Formula First

## Core Features:

- Formula Input: Allows users to input formulas and expressions using a custom editor with syntax highlighting and error checking.
- Dynamic Calculation: Evaluate the input formula in real-time, automatically updating results as values change.
- Unit Conversion: Automatically identify units in a formula and offer conversions (e.g., meters to feet) using a tool with a knowledge base of units.
- Formula Library: Maintains a library of common formulas (e.g., physics, math, finance) that users can browse, search, and import into their calculations.
- Graph Plotting: Generates 2D or 3D plots of formulas, allowing users to visualize the results.
- Calculation History: Saves previous calculations for users to review, modify, or reuse.
- Export: Export current calculation to one of the most popular formats like .txt, .pdf and .csv

## Style Guidelines:

- Primary color: Deep purple (#673AB7), evokes intelligence and sophistication.
- Background color: Light gray (#F5F5F5), provides a clean and unobtrusive backdrop.
- Accent color: Teal (#009688), creates contrast for interactive elements.
- Body font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for body text
- Headline font: 'Space Grotesk', a proportional sans-serif with a computerized, techy, scientific feel; use for headlines.
- Use minimalist line icons related to mathematical and scientific concepts.
- Emphasize clarity with a well-organized, grid-based layout.
- Use subtle animations to indicate loading or dynamic calculation updates.
